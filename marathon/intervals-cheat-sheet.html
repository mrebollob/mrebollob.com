Intervals.icu Workout Cheat Sheet

Quick reference for creating structured workouts in Intervals.icu. Workouts are written as plain text with a simple, readable syntax.

Based on the official documentation: Workout Builder Guide

Basic Syntax

Element Description
- Step text Start each workout step with a dash (-)
Section name Lines without a dash become section headers (Warmup, Main set, etc.)
6x Add repeats by including "Nx" before a set of steps
Press lap End step when lap button pressed (Garmin devices)

Duration Formats

Format Example Description
Xs 30s Seconds
Xm 10m Minutes
XmYs 1m30 or 1m30s Minutes and seconds
Xh 1h Hours

Distance Units

Distance-based workouts are supported for running and other sports.

1km or 1 km
1mi or 1 mile
400mtr or 400 meters
100yrd or 100 yards
Note: The letter "m" alone means minutes, not meters. Use mtr or meters for meter distances.

Intensity: Zones & Percentages

Format Example Description
X% 80% Percentage of FTP (cycling) or threshold pace (running)
X-Y% 70-80% Percentage range
X% HR 75% HR Percentage of max heart rate
X% LTHR 90% LTHR Percentage of lactate threshold HR
ZX Z2 Training zone (power/pace)
ZX HR Z3 HR Heart rate zone
ZX Pace Z4 Pace Pace zone

Absolute Pace (Running)

Specify exact pace values instead of percentages. Added November 2025.

Format Example Description
X:XX/km Pace 5:30/km Pace Pace per kilometer
X:XX/mi Pace 8:00/mi Pace Pace per mile
X:XX-Y:YY/km Pace 5:30-5:00/km Pace Pace range (slower to faster)
X:XX/400m Pace 1:45/400m Pace Pace per 400m (track)
X:XX/100m Pace 1:40/100m Pace Pace per 100m (swimming)
/km per kilometer
/mi per mile
/100m per 100 meters
/400m per 400 meters
/500m per 500 meters
/100y per 100 yards
/250m per 250 meters

Ramps & Cadence

Format Example Description
Ramp X-Y% Ramp 60-80% Gradual increase in intensity
X rpm 90 rpm Target cadence (cycling)
X-Y rpm 85-95 rpm Cadence range

Running Workout Examples

Easy Recovery Run Recovery
Easy Recovery Run - 5km 65-70% Pace

Simple 5km run at recovery pace (65-70% of threshold pace).

Tempo Run with Warmup/Cooldown Tempo
Warmup - 2km 65% Pace Tempo - 5km 88-92% Pace Cooldown - 2km 60% Pace

Classic tempo workout with proper warmup and cooldown. The tempo section is at lactate threshold pace.

VO2max Intervals (5x1000m) Intervals
Warmup - 2km 65% Pace - 4x 100mtr strides at 95% Main set 5x - 1km 95-100% Pace - 400mtr recovery at 55% Pace Cooldown - 2km 60% Pace

Classic VO2max workout: 5 repetitions of 1km at near-threshold pace with 400m recovery jogs.

Track Workout (400m Repeats) Intervals
Warmup - 10m 65% Pace Track Intervals 8x - 400mtr 1:35/400m Pace - 200mtr recovery at 60% Cooldown - 10m 60% Pace

Track session with 8x400m at specific pace target (1:35 per 400m) with 200m recovery jogs.

Marathon Pace Long Run Long Run
Long Run with Marathon Pace - 6km 70% Pace warmup - 16km 5:40/km Pace marathon pace - 3km 65% Pace cooldown

25km long run with 16km at marathon goal pace (5:40/km). Great for race-specific preparation.

Progression Run Tempo
Progression Run - 3km 70% Pace easy start - 3km 75% Pace moderate - 3km 80% Pace tempo - 3km 85% Pace strong finish - 1km 60% Pace cooldown

Gradually increase pace throughout the run. Great for building race instincts and negative split practice.

Fartlek Session Intervals
Warmup - 15m 65% Pace Fartlek 6x - 2m 90% Pace hard effort - 3m 70% Pace easy jog Cooldown - 10m 60% Pace

Unstructured speed play with alternating hard and easy efforts based on time rather than distance.

Hill Repeats Intervals
Warmup - 15m 65% Pace include dynamic stretches Hill Repeats 8x - 90s 95% uphill hard effort - 2m jog down recovery at 50% Cooldown - 10m 60% Pace

Strength-building hill workout. Focus on form during the uphill efforts, easy jog back down for recovery.

Threshold Run (Absolute Pace) Tempo
Threshold Session - 2km 6:00/km Pace warmup - 20m 4:50-4:40/km Pace threshold effort - 2km 6:30/km Pace cooldown

20-minute threshold run at specific pace targets. Using absolute pace values instead of percentages.
